# data_analysis_java

利用java操作txt、csv、excel文件，及简单数据分析

- csv/test/Demo.java

  包含：读取本地`csv`文件、移除指定列、选取指定列、获取列数据类型、按指定列排序、更改列名、数据打印

- csv/test/Demo2.java

  包含：手动创建`DataFrame`、常用统计项、数据打印、保存`csv`文件
  

- xls/test/Demo.java

  包含：读取本地`xls`文件、移除指定列、选取指定列、更改列名、自定义排序、数据打印
