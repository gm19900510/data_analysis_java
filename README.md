# data_analysis_java

利用java操作txt、csv、excel文件，及简单数据分析

- Demo.java

  包含：读取本地csv文件、移除指定列、选取指定列、获取列数据类型、按指定列排序、更改列名、数据打印

- Demo2.java

  包含：手动创建DataFrame、常用统计项、数据打印、保存csv文件
